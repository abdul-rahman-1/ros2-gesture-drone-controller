# ROS2 Resource Index Directory (`resource/`)

This directory contains ROS2 package index marker files required by `ament_python` build system.

---

## File

* **`imu_serial_bridge`**: Marker file created for ROS2 package discovery and `ament` index registration. Allows ROS2 CLI tools (`ros2 pkg`, `ros2 run`, `ros2 launch`) to find and locate `imu_serial_bridge` in the workspace.
